Set a SHP object's projection.

Notes
If the SHP object has a projection, and it is not &lt;define&gt;IPJ_TYPE_NONE&lt;/define&gt;, then it will be output to a file with the .prj extension when the first object is output. This function should be called BEFORE the first object is written.
