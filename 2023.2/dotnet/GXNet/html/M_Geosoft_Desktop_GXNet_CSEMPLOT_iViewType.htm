Remarks
First checks the view name to see if it starts with "XYplt_" or "Triplt_". Failing that it looks in the view REG for a value for "Components.Type", which will be either "XYPlot" or "TriPlot". If the view does not appear to be an XYPlot or a TriPlot view, the function returns &lt;define&gt;SEMPLOT_PLOT_UNKNOWN&lt;/define&gt;.
