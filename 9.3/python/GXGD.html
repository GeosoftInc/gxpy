GXGD class¶

class GXGD(wrapper=None)¶

GXGD class.

This class provides access to Geosoft grid files using an old interface. Only the GXDU.sample_gd function uses this class. Use the GXIMG class instead.

classmethod create(name, type)¶

This method creates a GXGD object.

Parameters:
  • name (str) – Name of the Grid File
  • type (int) – GD_STATUS
Returns:

Handle to the GXGD object
