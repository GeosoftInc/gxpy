GXACQUIRE class¶

class GXACQUIRE(wrapper=None)¶

GXACQUIRE class.

This class is used to import acQuire data. It uses the public acQuire API.

import_hole(proj, dir, para, geo_vv, delete, convert)¶

Import Drillhole data acQuire database into a GDB

Parameters:
  • proj (str) – Project name
  • dir (str) – Project directory
  • para (str) – Parameter File
  • geo_vv (GXVV) – List of geology name database
  • delete (int) – 0: Write to existing databases (overwrite holes), 1: Delete existing databases.
  • convert (int) – Convert Negatives (0,1)
Returns:

0 - Ok 1 - Error (Will not stop GX)
Return type:

int

New in version 6.0.1.

Note:

Point data and polygon data are saved into Dnnn lines in GDB, nnn representing incremental number starting from 0

import_point(db, para, convert)¶

Import Point Sample data acQuire database into a GDB

  • para (str) – Parameter File
  • convert (int) – Convert Negatives (0,1)

0 - Ok 1 - Error (Will not stop GX)

int

New in version 6.0.1.

Note:

Data existing in the receiving GDB file will be over-written. Point data and polygon data are saved into Dnnn lines in GDB, nnn representing incremental number starting from 0
