GX3DN class¶

class GX3DN(wrapper=None)¶

GX3DN class.

This class manages the rendering of a 3D view. It allows the positioning of the camera, specification of the zoom as well as some rendering controls for the axis. It is directly related to the GXMVIEW class.

set_scale(x, y, z)¶

Set the axis relative scales.

Parameters:
  • x (float) – X Scale (default 1.0)
  • y (float) – Y Scale (default 1.0)
  • z (float) – Z Scale (default 1.0)

New in version 6.1.

Note:

By default all scales are equal (1.0). By setting these scales, relative adjustments to the overall view of the 3D objects can be made. Note that they are relative to each other. Thus, setting the scaling to 5,5,5 is the same as 1,1,1. This is typically used to exaggerate one scale such as Z (1,1,5).
