Remarks
Works along with the IGetFileFilter_GUI function. Note that most values of FILE_FILTER_XXX will return &lt;define&gt;GS_DIRECTORY_NONE&lt;/define&gt;, and give the current workspace directory. This function is useful for constuction open/save dialog file filters, especially in GX.Net functions.
