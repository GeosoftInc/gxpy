vv
Type: GeoEngine.Core.GXNetXCVV
Split VV (REAL) (relative sizes of each profile window)
Remarks
The input VV values are the fractional heights for each profile window. Values are summed, and normalized (so you can enter "1,1,1", with a VV of length 3, if you want 3 equal profile windows). VV values beyond the maximum number of displayable profiles (&lt;define&gt;MAX_PROF_WND&lt;/define&gt;) are ignored.
