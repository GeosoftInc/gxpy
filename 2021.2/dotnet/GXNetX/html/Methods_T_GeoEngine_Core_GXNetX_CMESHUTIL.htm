  NameDescription
ApplyTransformation
Applies a transformation to a surface, see &lt;define&gt;SURFACE_TRANSFORMATION_METHOD&lt;/define&gt;for available operations. The existing mesh will be preserved, and a new mesh will be created with the target name in the target file. Reprojection willbe handled automatically in the case that the coordinate systems differ.
