CGXNETCoreiCheckError Method
Test if Geosoft function has terminated with an error

Namespace: GeoEngine.Core.GXNet
Assembly: geoengine.core.gxnet (in geoengine.core.gxnet.dll)
Notes
For Geosoft functions that do not return an error condition (all of them except those specifically designed for DLL use) you must call this function to test if an error has occured in the Geosoft function. If an error has occured, just clean-up and return. Error messages will have been registered by the Geosoft function.
