Launch the coordinate system definition/display GUI.

Available since Oasis montaj version: 5.0
Limitations: May not be available if running outside of a Oasis Montaj or from a command line program.  
License: Available to anyone.  

Namespace: Geosoft.Desktop.GXNet
Assembly: geosoft.desktop.gxnet (in geosoft.desktop.gxnet.dll)
Notes
Launches the new GX.Net single-dialog coordinate system definition dialog. The input IPJ is modified on return if OK is selected (and the editable parameter is 1). The "Data source label" and "Data source" is information displayed in the dialog for the user to know where the IPJ came from (e.g. "Grid: X.grd")
