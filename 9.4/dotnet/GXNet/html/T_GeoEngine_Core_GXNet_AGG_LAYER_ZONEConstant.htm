SHADE
If &lt;define&gt;AGG_LAYER_ZONE_SHADE&lt;/define&gt; is specified, a shaded relief layer is created from the specified grid. A new grid file will also be created to hold the shaded relief image data. This file will have the same name as the original grid but with "_s" added to the root name. It will always be located in the workspace directory regardless of the location of the original source image. If the file already exists, it will used as it is. Shading is always at inclination = declination = 45 deg. with default scaling. If different shading is desired, use the LayerShadeIMG_AGG method.
