geosoft.gxpy.va submodule¶

Geosoft vector arrays (vector of array elements). VA and VV classes are related based on a key called a fiducial, which has a start value and increment between values. The refid() method can be used to resample vector data to the same fiducial so that vector-to-vector operations can be performed.

See also

geosoft.gxpy.vv, geosoft.gxapi.GXVA, geosoft.gxapi.GXVV
