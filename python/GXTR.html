GXTR class¶

class GXTR(handle=0)¶

GXTR class.

The GXTR object contains trend information about a grid or grid pager. Currently, it is used only in conjunction with the GXIMG.get_tr, GXIMG.set_tr, and GXPGU.trend functions.
