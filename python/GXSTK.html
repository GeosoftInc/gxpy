GXSTK class¶

class GXSTK(handle=0)¶

GXSTK class.

The GXSTK class is used for plotting a single data profile in an GXMVIEW. The GXMSTK class (see GXMSTK) is used to plot multiple GXSTK objects to a single map.

Use GXMSTK.add_stk fuction to create a GXSTK object before using functions in this file

SEE GXMSTK FILE FOR DETAILED DESCRIPTIONS OF ALL FUNCTION PARAMETERS.
