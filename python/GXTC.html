GXTC class¶

class geosoft.gxapi.GXTC¶
The TC object is used in gravitational modelling to create
a terrain correction grid from a topography grid. This is
accomplished with a call first to geosoft.gxapi.GXTC.grregter(), which determines
the terrain correction from an input topography grid, then
to geosoft.gxapi.GXTC.grterain(), which calculates the actual corrections at
the input positions.
