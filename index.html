Geosoft GX API Reference Documentationn

The Python reference documentation for the Geosoft GX API can be found here.

The complete GX Developer programming guide can be found in a separate on-line document: GX Developer Guide. The programming guide is available as a separate version for each released version of GX Developer. Refer to the documentation version that matches the API version you wish to use.
