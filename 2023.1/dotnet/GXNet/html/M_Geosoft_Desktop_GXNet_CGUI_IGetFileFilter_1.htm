Remarks
Returns the four parts of the file filter; e.g. for &lt;define&gt;FILE_FILTER_GDB&lt;/define&gt; it returns: Filter: "Database (*.gdb)" Mask: "*.gdb" Extension: "gdb" Directory: "&lt;define&gt;GS_DIRECTORY_NONE&lt;/define&gt;" This function is useful for constuction open/save dialog file filters, especially in GX.Net functions.
