The &lt;define&gt;ITR_ZONE_DEFAULT&lt;/define&gt; model will ask the IMG to provide a model if it can. If a shaded relief model is selected, a shaded image will be created and a shaded image file will be created with the same name as the original grid but with the suffux "_s" added to the name part of the grid.
