Creates an ASCII file to write to.

Available since Oasis montaj version: 6.2
License: Available to anyone.  

Namespace: GeoEngine.Core.GXNet
Assembly: geoengine.core.gxnet (in geoengine.core.gxnet.dll)
Notes
Before version 6.2. text in on the GX API level were handled as characters in the current ANSI code page defining how characters above ASCII 127 would be displayed. 6.2. introduced Unicode in the core montaj engine that greatly increased the number of symbols that can be used. The &lt;define&gt;WA_ENCODE&lt;/define&gt; constants were introduce that controls how text are written to files on disk with the WA class.
