Notes
The ITR values are interpreted as cumulative percent values, using the "PERCENT=1" value in the ITR's REG. Note that processes using ITRs do not automatically know to convert between percent values and "actual" data values. The REG "PERCENT" value is simply a flag to indicate to a user that the values are intended to be in the range from 0 < x < 100. The ITR should not, therefore, be applied directly to data unless that data is already given in percent. If the file name is defined on input, the initial ITR will be loaded from it. If it is left blank, a default 5-color transform with The color transform file name is used as the default when the save button is pushed, and is updated both after the load and save buttons are pushed by the value input or selected by the user.
