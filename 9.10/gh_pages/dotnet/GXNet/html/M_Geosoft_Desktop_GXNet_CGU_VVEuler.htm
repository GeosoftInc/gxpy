Get Euler solutions of depth from VVs and grids.

Available since Oasis montaj version: 5.0
License: Available to anyone with an Oasis Montaj license.  

Namespace: Geosoft.Desktop.GXNet
Assembly: geosoft.desktop.gxnet (in geosoft.desktop.gxnet.dll)
Notes
All VVs must be REAL The output X and Y values are the same as the inputs, except if &lt;define&gt;PEAKEULER_XY_FIT&lt;/define&gt; is selected. All other output values are set to dummy if: a) The input X or Y is a dummy b) The derived window size is a dummy. c) The derived solution is outside the range d) The solution is invalid (singular matrix)
